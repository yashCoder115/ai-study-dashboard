# ai-study-dashboard
AI Study Dashboard – A modern web app that summarizes notes, extracts text from PDFs and images, and helps students study smarter with a clean interactive UI.
AI Study Dashboard is a modern web-based productivity tool designed for students. It allows users to quickly summarize notes, extract text from PDFs and images, and organize study material in a simple interactive interface.

Features:
• Text summarization from pasted notes
• PDF text extraction and summarization
• Image OCR to extract text from screenshots or photos
• Word counter for study notes
• Copy summary feature
• Dark / Light mode toggle
• AI-style typing animation
• Drag and drop file upload
• Animated background UI

Technologies Used:
HTML
CSS
JavaScript
PDF.js
Tesseract.js

This project was built as part of my learning journey in Artificial Intelligence and Web Development.
